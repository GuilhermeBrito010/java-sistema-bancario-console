# Sistema Bancário em Java (Console)

Projeto simples em Java para praticar Programação Orientada a Objetos (POO),
uso de Scanner, estruturas de decisão e repetição.

## Funcionalidades
- Login com usuário e senha
- Menu interativo
- Ver saldo
- Depositar
- Sacar
- Validações de entrada

## Tecnologias
- Java

## Objetivo
Projeto desenvolvido para fins de estudo, focado em POO básica e lógica de programação.
